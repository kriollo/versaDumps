## Project Overview

**VersaDumps** is a cross-platform desktop debugging tool built with [Wails v2](https://wails.io/) (Go backend + Vue 3 frontend). It acts as a visual dump receiver: backend applications POST JSON to `localhost:9191/data`, and the desktop app displays the payloads in real-time. It also monitors log files using fsnotify.

Current version: **3.1.0**

## Commands

All commands are run from the `app/` directory unless noted.

### Development

```bash
cd app
wails dev          # Start hot-reload dev server (Go + Vue simultaneously)
```

### Build

```bash
cd app
wails build                                              # Current platform
wails build -platform linux/amd64                        # Linux
wails build -platform windows/amd64 -nsis -webview2=embed  # Windows (NSIS installer)
wails build -platform darwin/amd64                       # macOS
```

### Tests

```bash
# Go tests (from app/)
go test ./... -v
go test ./... -coverprofile=coverage.out -cover
go tool cover -html=coverage.out -o coverage-go.html

# All tests via script (from app/)
./run-tests.sh      # Linux/macOS
./run-tests.ps1     # Windows PowerShell
```

Frontend test files exist under `frontend/src/components/__tests__/` and `frontend/src/i18n/__tests__/` using Vitest (see `run-tests.sh` for invocation details).

## Architecture

### IPC Bridge (Go ↔ Vue)

Wails binds exported Go methods on the `App` struct directly to the frontend. The auto-generated JS bindings live in `frontend/wailsjs/go/main/` — **do not edit these manually**; they are regenerated by `wails dev`/`wails build`.

- **Go → Frontend**: `runtime.EventsEmit(ctx, "eventName", payload)` — Vue subscribes via `window.runtime.EventsOn()`
- **Frontend → Go**: `window.App.MethodName(args)` (promise-based) — any exported method on `App` struct in `app.go`

Key emitted events: `newData`, `logUpdated`, `configLoaded`, `logMessage`, `updateAvailable`

### Backend (Go, `app/`)

| File | Role |
|------|------|
| `main.go` | Wails init, embeds frontend assets via `//go:embed` |
| `app.go` | `App` struct with lifecycle hooks (`startup`, `domReady`, `beforeClose`), all exported IPC methods |
| `server.go` | HTTP server — `GET /health` and `POST /data` (10 MB limit, 30s timeout) |
| `config.go` | YAML config load/save, profile management, migration logic |
| `logwatcher.go` | fsnotify-based log monitoring — incremental tail, file rotation detection, level/extension filters |
| `updater.go` | GitHub release API version check and download |
| `updater_unix.go` / `updater_windows.go` | Platform-specific update installation |
| `badge_*.go` | Platform-specific taskbar badge (shows dump count) |

### Frontend (Vue 3 + Tailwind, `app/frontend/src/`)

No external state library — Vue 3 Composition API with reactive `ref`s in `App.vue`. State flows down via props; events bubble up via `emit`.

| Component | Role |
|-----------|------|
| `App.vue` | Root — split-view layout, all IPC event subscriptions |
| `ConfigModal.vue` | Settings dialog, profile CRUD |
| `LogItem.vue` | Single HTTP dump entry with expand/collapse |
| `JsonTreeView.vue` + `JsonTreeViewNode.vue` | Recursive JSON explorer |
| `LogFileViewer.vue` | Real-time log file display |
| `LogFoldersManager.vue` | Add/edit/delete log folder monitors |
| `UpdateNotification.vue` | Update available banner |
| `i18n/` | Custom i18n (no external lib) — `en.js` and `es.js` |

### Configuration

- **Location**: `~/.config/VersaDumps/config.yml` (Linux/macOS) / `%APPDATA%/VersaDumps/config.yml` (Windows)
- Supports multiple **named profiles**, each with its own server host/port, theme, language, and log folder monitors
- Config is loaded at startup and saved via `SaveConfig()` IPC call from frontend

### HTTP Data Protocol

Backend apps send dumps via:
```
POST http://localhost:9191/data
Content-Type: application/json

{ "any": "json", "payload": true }
```

The server validates JSON, then emits a `newData` event with the raw payload to the frontend.
